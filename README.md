# Singleplayer-Chess

由于一些显而易见的原因，这玩意不支持在线多人（至少到现在）。如果你要二人对弈，可以让两个人轮流~~机惨~~操作！

chess.exe 是已经被编译完成的 chess.cpp。你可以直接下载 chess.exe 而不是下载 chess.cpp 以获得更好的体验！（p.s. chess.exe 使用如下指令编译：`g++.exe chess.cpp -o chess.exe -g3 -fno-ms-extensions -Ofast -std=gnu++2a -pipe -Werror=return-type -Werror=vla -D_DEBUG -lws2_32 -Wl,--stack,1073741824`）
